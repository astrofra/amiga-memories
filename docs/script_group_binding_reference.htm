Group Binding Reference
1. Group management functions
GroupAddItemvoid: Group, Item
Add an item to an item group.
GroupFindItemItem: Group, string name
Find an item in group from its name.
GroupGetItemListarray: Group
Return the group item list.
GroupGetRootItemItem: Group
Return the root item of a group.
GroupItemIsMemberbool: Group, Item
Returns true if item the belongs to the group, false otherwise.
GroupRemoveItemvoid: Group, Item
Remove an item from an item group.
GroupRenderSetupvoid: Group, RenderResourceFactory
Setup rendering data for all members of a group.
GroupResetvoid: Group
Reset all members of a group.
GroupSetInvisiblevoid: Group, bool
Hide or show all members of a group.
GroupSetNamevoid: Group, string
Set the name of an item group.
GroupSetRootItemvoid: Group, Item
Set the root item of a group. All members of a group are linked to its root item.
GroupSetupvoid: Group
Setup all members of a group.
GroupSetupScriptvoid: Group
Setup the script object of all members of a group.
2. Group transformation functions
GroupOffsetMatrixvoid: Group, matrix
Apply a 4x4 offset matrix to all group members.
3. Group motion functions
GroupSetMotionAnimationsourcegroup: Group, string name, float blend
Set motion on all group items, specify the blend duration, stop all current animation sources.
