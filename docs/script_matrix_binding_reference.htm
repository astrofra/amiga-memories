Matrix Binding Reference
1. Global functions
EulerFromDirectionVector: Vector direction
Convert a world space direction vector to Euler angles.
MatrixToEulerVector: Matrix3, RotationOrder
Convert a world space matrix to euler angles.
RotationMatrixFromDirectionMatrix3: Vector direction
Convert a world space direction vector to a 3x3 rotation matrix.
RotationMatrixXVector: Matrix3, float angle
Create a 3x3 rotation matrix around the X axis, angle is in degree.
local m = RotationMatrixX(Deg(45))
RotationMatrixYVector: Matrix3, float angle
Create a 3x3 rotation matrix around the Y axis, angle is in degree.
local m = RotationMatrixY(Deg(45))
RotationMatrixZVector: Matrix3, float angle
Create a 3x3 rotation matrix around the Z axis, angle is in degree.
local m = RotationMatrixZ(Deg(45))
TransformationMatrixMatrix4: Vector position, Vector euler, Vector scale, Vector pivot
Create a position, rotation, scale, offset 4x4 matrix.
The offset is applied first, scale second, rotation third and finally position is applied.

GetRowVector: int index

SetRowvoid: int index, Vector row
