Physic Binding Reference
1. Constraint management functions
ConstraintEnablevoid: Constraint, bool
Enable/disable constraint.
SceneAddPointConstraintConstraint: Scene, string name, Item a, Item b, Vector pivot_a, Vector pivot_b
Create a new point constraint between two items. The pivot position is in item space. The null item can be set as item B to specify an unmovable world space hook.
2. Constraint configuration functions
ConstraintGetItemItem: Constraint
Get constraint item.
ConstraintGetPivotAVector: Constraint
Get the constraint item A pivot vector.
ConstraintGetPivotBVector: Constraint
Get the constraint item B pivot vector.
ConstraintSetItemAvoid: Constraint, Item
Set constraint item A.
ConstraintSetItemBvoid: Constraint, Item
Set constraint item B.
ConstraintSetPivotAvoid: Constraint, Vector pivot
Set the constraint item A pivot vector.
ConstraintSetPivotBvoid: Constraint, Vector pivot
Set the constraint item B pivot vector.
