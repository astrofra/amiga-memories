1. Handshaking functions
PeerNetOpenClientbool: string ip, int port
Open a client connection to a given ip address and port.
PeerNetOpenServerbool: string ip, int port
Open a listening server connection on a given ip address and port. Note: A server connection is usually opened on the local host ip "127.0.0.1".
2. Networking functions
PeerNetUpdatevoid:
Update the peer network interface, dispatches pending events and call their associated callbacks. This function must be called frequently to refresh the interface and prevent connection loss.
