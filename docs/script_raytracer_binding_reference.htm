Raytracer Binding Reference
1. Generic functions
NewRaytracerRaytracer: GraphicResourceFactory
Create a new raytracer instance.
RaytracerSetAntialiasvoid: raytracer, bool enable, int sample_count, float threshold, bool jitter
Set the raytracer antialias output. The number of AA sample can be set and jitered AA selected. Default threshold: 0.015.
RaytracerSetGlobalIllumvoid: raytracer, bool enable, int sample_count, int max_indirect_bounce
Enable the raytracer global illumination. The number of GI sample can be set as well as a maximum number of indirect bounce.
RaytracerSetInterlacevoid: raytracer, bool interlaced, bool even_frame
Set the raytracer interlace output. The frame parity is specified as the second parameter.
RaytracerSetScenevoid: raytracer, Scene
Set raytracer scene, this function might be a little slow as it setups a lot data to work with.
RaytracerStartInterlacedSequencevoid: raytracer
Start an interlaced sequence. This function should be called when beginning a new sequence in order to ensure correct frame parity.
RaytracerTracevoid: raytracer, string out_picture, int width, int height
Raytrace current scene to a Targa picture file. Note that when rendering interlaced sequences one of two calls to this function will return false to notify you that the current frame was only buffered and not saved.
