1. Channel management functions
MixerChannelGetStateChannelState: Mixer, int channel
Return the channel state.
MixerChannelPausevoid: Mixer, int channel
Pause playback on a given channel.
MixerChannelResumevoid: Mixer, int channel
Resume playback on a given channel.
MixerChannelStopvoid: Mixer, int channel
Stop playback on a given channel.
MixerChannelStopAllvoid: Mixer
Stop replay on all mixer channels.
2. Mixer control functions
MixerMutevoid: Mixer
Mute all sound output.
MixerSetGainvoid: Mixer, float gain
Set the global mixer gain.
MixerSoundStartChannel: Mixer, Sound
Play a sound on the first channel available, the channel used for replay is returned. If no free channel was found -1 is returned.
MixerSoundStartFastbool: Mixer, Sound
Play a sound on the first channel available, this function returns immediately and the channel used for replay is not returned.
MixerStreamStartChannel: Mixer, uri
Play a stream on the first channel available, the channel used for replay is returned. If no free channel was found -1 is returned.
MixerStreamStartFastbool: Mixer, uri
Play a stream on the first channel available, this function returns immediately and the channel used for replay is not returned.
MixerUnmutevoid: Mixer
Unmute all sound output.
3. Channel control functions
MixerChannelLockChannel: Mixer
Lock a new mixer channel.
MixerChannelSetGainvoid: Mixer, Channel, float gain
Set channel gain. Note: The channel gain is not reset by starting new sounds.
MixerChannelSetLoopModevoid: Mixer, Channel, ChannelLoop loop
Set channel loop mode.
MixerChannelSetLoopPositionvoid: Mixer, Channel, int ms
Set channel loop position in millisecond, the loop mode must be set to LoopRepeat for this setting to have any effect.
MixerChannelSetPanningvoid: Mixer, Channel, float panning
Set channel panning (Default: 0.5). Note: The channel panning is not reset by starting new sounds.
MixerChannelSetPitchvoid: Mixer, Channel, float pitch
Set channel pitch (Default: 1.0). Note: The channel pitch is not reset by starting new sounds.
MixerChannelStartbool: Mixer, Channel, Sound
Play a sound on a specific channel. Returns true on success, false otherwise.
MixerChannelStartStreambool: Mixer, Channel, string uri
Play a stream on a specific channel. Returns true on success, false otherwise.
MixerChannelUnlockvoid: Mixer, Channel
Unlock channel.
MixerChannelUnlockAllvoid: Mixer
Unlock all channels.
