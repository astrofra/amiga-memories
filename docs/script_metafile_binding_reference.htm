2. Tag functions
MetafileAddRootMetatag: Metafile, string tag_name
Add metafile root, returns the newly created metatag.
MetafileAddRootWithValueMetatag: Metafile, string tag_name, ...
Add metafile root, returns the newly created metatag.
MetafileGetTagMetatag: Metafile, string tag_path
Get metatag from metafile, the metatag path is formatted as follow: tag:child:tag_id; (eg: 'Pref:Sound:Volume;').
MetafileGetTypedTagMetatag: Metafile, string tag_path, TagType
Get metatag of a specific value type from a metafile.
MetatagAddChildMetatag: Metatag, string tag_name
Add metatag child, returns the newly created metatag.
MetatagAddChildWithValueMetatag: Metatag, string tag_name, ...
Add metatag child, returns the newly created metatag.
MetatagGetChildrenarray: Metatag
Return all children of a metatag in an array.
MetatagGetNamestring: Metatag
Get metatag name.
MetatagGetTagMetatag: Metatag, string tag_path
Find metatag child.
MetatagGetTypedTagMetatag: Metatag, string tag_path, TagType
Find child of a specific value type from a metatag.
MetatagGetValue...: Metatag
Get metatag value.
MetatagSetValuevoid: Metatag, ...
Set metatag value, the value type is automatically handled.
