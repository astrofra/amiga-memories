1. General project functions
ProjectEndvoid: Project
Exit project.
ProjectGetClockClock: Project
Get the project clock object.
ProjectGetScriptInstanceInstance: Project
Get the project script instance.
2. UI Font management functions
ProjectDeleteUIFontAliasvoid: Project, string alias
Delete a font alias from the font cache.
ProjectGetUIFontFont: Project, string name
Retrieve a font from its name.
ProjectLoadUIFontFont: Project, string path
Load a TrueType font in the project font cache.
ProjectLoadUIFontAliasedFont: Project, string path, string alias
Load a TrueType font under a specific alias.
3. Layer management functions
ProjectAddLayerProjectlayer: Project, Projectscene, float zorder
Create a new layer to display a given project scene.
ProjectLayerGetSceneProjectscene: Projectlayer
Return the scene instance this layer is displaying.
ProjectLayerGetZOrderfloat: Projectlayer
Get project layer Z order.
ProjectLayerSetZOrdervoid: Projectlayer, float
Set project layer Z order.
ProjectSetAll2DLayerOffsetvoid: Project, float x, float y
Set the offset of all 2d layer in the current project stack. The offset is specified in the layer reference resolution.
4. Scene management functions
ProjectFindSceneProjectscene: Project, string path
Returns an instance of a specific scene in the project, returns an invalid object if no such instance could be found.
ProjectGetSceneLayerListarray: Project, Projectscene
Return all the project layers displaying a specific scene instance. A scene might be displayed by several different layers (eg. in order to display a split-screen view).
ProjectInstantiateSceneProjectscene: Project, string path
Instantiate a project scene, do not forget to create a layer to display it.
ProjectNewScene2DScene: Project
Create a new scene 2D.
ProjectNewScene3DScene: Project
Create a new scene 3D.
ProjectSceneActivatevoid: Project, Projectscene, bool
Activate or deactivate a project scene. When deactivated a scene is not updated and displayed anymore.
ProjectSceneGetInstanceScene: Projectscene
Get a project scene instance object (might be Scene, Scene2d or UI).
ProjectSceneIsActivebool: Project, Projectscene
Return the active state of a project scene.
ProjectSceneSetGlobalvoid: Projectscene
Register scene as the global scene script object (g_scene). Note: The project changes the global scene before performing any operation on a scene already, you should rarely need this function.
ProjectUnloadScenevoid: Project, Projectscene
Unload a project scene from memory. Note that only the scene graph, motions and groups are dropped. Engine cached resources are left untouched.
