Script Binding Reference
1. Debugging functions
__Assertvoid: bool, string
Assert an expression, throw an engine exception and suspend the VM if not verified.
2. Generic functions
ObjectIsSamebool: Object, Object
Returns true if two given engine object references are pointing to the same object.
ObjectIsValidbool: Object
Returns true if the given engine object is valid, false otherwise.
SQAbsModint: int value, int divider
Returns integer modulo of the absolute of a given value.
SQModint: int value, int divider
Returns integer modulo of a value.
SQPowfloat: float value, float pow
Returns value raised to power.
