Geometry Binding Reference
1. Generic functions
GeometryComputeIsoSurfacevoid: Item item, int nb_metaball, array pos_metaball, array value_metaball
Create an iso surface to the geometry with the array of metaball.
GeometryGetNamestring: Geometry geo
Get geometry name.
GeometryOptimizevoid: Geometry geo
Optimize geometry for realtime rendering.
GeometrySetHiddenvoid: Geometry geo, bool hidden
Hide or show geometry, items referring to this geometry will still be updated.
GeometrySetupvoid: Geometry geo
Setup geometry in the renderer.

Get material from name.
GeometryGetMaterialCountint: Geometry geo
Get geometry material count.
GeometryGetMaterialFromIndexMaterial: Geometry geo, int index
Get material from index.
GeometryGetMaterialListarray: Geometry geo
Return a list of all materials in a geometry.
