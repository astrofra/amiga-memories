1. Clock functions
SceneGetClockScalefloat: Scene
Get engine clock scale.
SceneResetClockvoid: Scene
Reset engine clock.
SceneSetClockScalevoid: Scene, float
Set engine clock scale. All clock dependent components in the engine are affected by this scale.
SceneSetFixedDeltaFramevoid: Scene, float
Force a constant fixed delta frame value. Note: All clock dependent components will become frame rate dependent.
2. Motion functions
SceneSetMotionAnimationsourcegroup: Scene, string name, float blend
Set motion on all scene items, specify the blend duration, stop all current animation sources.
SceneSetMotionAddAnimationsourcegroup: Scene, string name, float blend
Set motion on all scene items, specify the blend duration, current sources are not stopped.
3. I/O functions
SceneLoadbool: Scene, string
Append scene content to a scene object. Returns true on success, false otherwise.
SceneLoadAndStoreGroupGroup: Scene, string, ImportFlag
Append scene content to a scene object, store all appended content in a group.
ScenePreloadResourcesbool: Scene, RenderResourceFactory, string
Load all scene geometries and dependencies. As long as the render factory cache is not explicitly emptied, all subsequent load request will return a cached instance.
SceneSavebool: Scene, string
Save scene content to file. Returns true on success, false otherwise.
4. Management functions
SceneAddCameraCamera: Scene, string name
Create a new camera.
SceneAddLightLight: Scene, string name
Add a new light to scene.
SceneAddObjectObject: Scene, string name
Add a new object to scene.
SceneDeleteAllLightsvoid: Scene
Delete all lights from scene.
SceneDeleteContentvoid: Scene
Delete all scene content.
SceneDeleteItemvoid: Scene, Item
Delete item from scene.
SceneDeleteLightvoid: Scene, Light
Delete light from scene.
SceneDeleteObjectvoid: Scene, Object
Delete object from scene.
SceneEnableItemDeletionQueuevoid: Scene, bool
Enable or disable the item deletion queue. Use SceneFlushDeletionQueue to manually empty the deletion queue.
SceneEndvoid: Scene
Set the end scene flag.
SceneFlushDeletionQueuevoid: Scene
Immediately delete all pending item deletion commands.
SceneGetCurrentCameraCamera: Scene
Get scene current camera.
SceneGetGroupListarray: Scene
Return an array containing all scene groups.
SceneGetPathvoid: Scene
Return the scene path.
SceneGetScriptInstanceInstance: Scene
Get scene script instance.
SceneGetScriptInstanceFromClassInstance: Scene, string class_name
Get scene script instance from class.
SceneGroupActivatevoid: Scene, Group, bool active
Set group member items active flag, effectively stops all physic, collision and display operations from happening on all group items.
SceneItemActivatevoid: Scene, Item, bool active
Set item active flag, effectively stops all physic, collision and display operations from happening.
SceneItemActivateHierarchyvoid: Scene, Item, bool active
Set item active flag on an item and its children.
ScenePushRenderablevoid: Scene, Renderer
Push the scene renderable onto a renderer render queue.
SceneRecordLocationvoid: Scene
For each item, record current transformation as the default transformation.
SceneRenderSetupvoid: Scene, Resourcefactory
Setup scene render resources.
SceneResetvoid: Scene
Reset a scene object. Note: All items are move back to their default transformation if recorded.
SceneSetCurrentCameravoid: Scene, Camera
Set camera as scene current camera.
SceneSetGlobalvoid: Scene
Set the scene as the script global object.
SceneSetItemDeletionDelayvoid: Scene, float
Set item deletion delay. Deprecated: Please use SceneEnableItemDeletionQueue.
SceneSetItemStaticvoid: Scene, Item, bool
Flag item as static, improves culling performance.
SceneSetRenderlessvoid: Scene, bool
Controls automatic rendering of the scene, if the scene is renderless it is up to the user to implement its rendering in the OnRender event.
SceneSetScriptvoid: Scene, string script_file, string script_class
Set the scene script file and class to instantiate.
SceneUpdatevoid: Scene
Update scene, run all logic, physic and call script callbacks.

6. Rendering functions

7. Debug/profile functions
ScenePrintItemListvoid: Scene
Dump scene item list to log.
8. Item management functions
SceneDuplicateItemvoid: Scene, Item
Duplicate an item and all its properties, returns the new instance.
You need to call SceneSetupItem() and ItemRenderSetup() on the new instance if you use this function at runtime.
SceneFindItemItem: Scene|Instance, string
Find an item in scene.
SceneFindItemChildvoid: Scene|Instance, Item parent, string child_name
Find child item of a given item in scene.
SceneGetItemChildrenListarray: Scene, Item
Returns an array of all the children items in the scene of a given parent item.
SceneGetItemListarray: Scene|Instance
Returns an array of all items in the scene.
SceneSetupItemvoid: Scene, Item
Setup an item. You need to call ItemRenderSetup() to setup the render object of this item.
9. Group management functions
SceneDeleteGroupvoid: Scene|Instance, Group
Delete a group and all its members from a scene.
SceneFindGroupGroup: Scene|Instance, string name
Find a group in scene from its name.
10. Physic functions
SceneFindConstraintConstraint: Scene, string name
Find a constraint in scene from its name.
SceneGetPhysicFrequencyfloat: Scene
Returns the physic system update frequency in Hertz.
ScenePhysicAllocateCollisionNodevoid: Scene, int max_node, int max_contact
Allocate physic system internal structures. A collision node is used for each collision occurring during the collision detection phase. Many contacts can be generated for each collision occurring.
ScenePhysicEnableDeactivationvoid: Scene, bool
Enable or disable physic system item deactivation.
SceneSetGravityvoid: Scene, Vector
Set default physic system gravity (Default: Earth gravity: 0.0, -9.8, 0.0). Note: This only affects the gravity for newly created items. In order to modify the gravity for a given item please use the item binding Physics functions.
SceneSetPhysicConstraintSolverIterationvoid: Scene, int
Set the number of iteration used by the physic constraint solver.
SceneSetPhysicContactSolverIterationvoid: Scene, int
Set the number of iteration used by the physic contact solver.
SceneSetPhysicFrequencyvoid: Scene, float
Set the physic system update frequency (default: 75 Hertz).
11. Collision functions
SceneCollisionRaytraceTraceResult: Scene, Vector from, Vector direction, int collision_mask, ShapeMask, float max_distance
Raytrace the collision system, collision shape type and collision ids can be filtered through bit mask usage.
SceneRaytraceTriggerListItem: Scene, Vector ray_origin, Vector ray_direction, float ray_length
Raytrace scene triggers, return the trigger item hit.
