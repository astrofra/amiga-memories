Global Reference
1. Global
beginPolygonvoid:
Begin declaration of a new polygon.
clearvoid:
Clear current template definition, keep the current material table.
clearMaterialvoid:
Push a material on the template material table.
endPolygonvoid: int material
End declaration of the current polygon and specify its material index.
instantiateGeometry: Engine engine, string name
Instantiate the current template as a named geometry. Note: If the geometry name is already found in the current engine cache, a cached copy will be returned instead of a new instantiation.
pushColorvoid: Vector
Push a color on the current polygon declaration.
pushMaterialvoid: string path
Push a material path on the template material table.
pushNormalvoid: Vector
Push a normal on the current polygon declaration.
pushUVvoid: int channel, Uv
Push an UV on a channel of the current polygon declaration.
pushVertexvoid: Vector
Push a vertex on the current polygon declaration.
setVertexMergeThresholdvoid: float threshold
Set the vertex merging algorithm threshold.
2. Miscellaneous platform functions
PlatformOpenURLbool: string id
Open an URL using the platform browser if available.
PlatformSendToBackgroundbool: bool kill
Send the current application to background, optionally try to kill it.
3. Platform visibility functions
PlatformOpenAppPagevoid:
Open the platform specific application page, use this to send the user to your rating page.
4. Licensing functions
PlatformLicensingCheckvoid: string license_info
Launch an asynchronous licensing request, the result will be sent to your script global 'OnLicensingEvent(String event)' function.
PlatformLicensingIsSupportedbool:
Returns true if the current platform supports app license verification.
5. In-app billing functions
PlatformBillingConfirmEventvoid: string id
Confirm a billing event (like a purchase or a refund).
PlatformBillingIsSupportedbool:
Returns true if the current platform supports in-app billing (in-app purchase).
PlatformBillingRequestPurchasevoid: string id
Launch a purchase intent for the specified virtual good identifier. Returns true of the intent was successfully launched. Results will be transmitted to you through the global 'OnBillingEvent(String event, String item)' script callback.
PlatformBillingRestorePurchasesvoid:
Restore all managed purchases.
PlatformGetLocalestring:
Return the current platform locale ISO2 code string (ISO 3166 two-character alphabetic code).
6. Analytics functions
PlatformAnalyticsLogEventvoid: string name
Log an analytics event.
PlatformAnalyticsServeFullscreenAdvoid: string location
Serve a fullscreen ad.
7. Host device functions
PlatformGetDeviceNamestring:
Return the device name the platform is currently running on.
