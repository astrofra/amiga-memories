Resource Factory Binding Reference
1. Resource factory general functions
ResourceFactoryPurgeint: Resourcefactory
Purge the resource factory cache.
2. Graphic resource functions
ResourceFactoryLoadPicturePicture: Resourcefactory, string name
Load a picture from a graphic resource factory.
3. Render resource functions
ResourceFactoryLoadGeometryTexture: Resourcefactory, string name
Load a geometry from a resource factory.
ResourceFactoryLoadGeometryExTexture: Resourcefactory, string name
Load a geometry from a resource factory, optionally bypassing the cache.
ResourceFactoryLoadMaterialMaterial: Resourcefactory, string name
Load a material from a resource factory.
ResourceFactoryLoadMaterialExMaterial: Resourcefactory, string name, bool bypass_cache
Load a material from a resource factory, optionally bypassing the cache.
ResourceFactoryLoadRasterFontRasterfont: Resourcefactory, string name
Load a raster font from a resource factory.
ResourceFactoryLoadTextureTexture: Resourcefactory, string name
Load a texture from a resource factory.
ResourceFactoryLoadTextureExTexture: Resourcefactory, string name
Load a texture from a resource factory, optionally bypassing the cache.
ResourceFactoryNewTextureTexture: Resourcefactory
Create a new texture from a resource factory.
4. Mixer resource functions
ResourceFactoryLoadSoundSound: Resourcefactory, string name
Load a sound from a resource factory.
