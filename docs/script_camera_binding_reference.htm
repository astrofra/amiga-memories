3. Viewport functions
CameraCullObjectVisibilityFlags: Camera camera_to_cull_against, Object object_to_cull
Cull an object against the camera frustum. Returns a visibility flag mask.
CameraCullPositionVisibilityFlags: Camera camera_to_cull_against, Vector world_position
Cull a position in world space against the camera frustum. Returns a visibility flag mask.
CameraGetFovfloat: Camera
Get camera fov in radian.
CameraScreenToWorldVector: Camera, Renderer, float x, float y
Transform a normalized screen space position to a world space position.
The screen space position is the result of a projection on the Z=1 imaginary plane, situated 1 meter in front of the camera.
CameraScreenToWorldPlaneVector: Camera, Renderer, float x, float y, float plane_distance_in_meters
Transform a normalized screen space position to a world space position.
The screen space position is the result of a projection on an imaginary plane situated in front of the camera.
CameraSetClippingvoid: Camera, float near, float far
Set camera near and far clipping planes.
CameraSetFocalDistancevoid: Camera, float distance_in_meters
Set camera focal distance, set to 0 or less to disable depth of field on the camera.
CameraSetFovvoid: Camera, float fov_in_radian
Set camera fov in radian.
CameraSetFStopvoid: Camera, float fstop_in_meters
Set camera f-stop, set to 0 or less to disable depth of field on the camera.
CameraWorldToScreenVector: Camera, Renderer, Vector world_position
Transform a world space position to a normalized screen space position, in the range { [0;1], [0;1] }.
