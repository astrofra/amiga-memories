3. Input device state functions
DeviceInputLastValuefloat: Inputdevice, DeviceInput
Returns the value of a device input during the last update.
DeviceInputValuefloat: Inputdevice, DeviceInput
Returns the current value of a device input.
DeviceIsKeyDownbool: Inputdevice, DeviceKey
Returns true if the given device key is currently down, false otherwise.
DeviceKeyPressedbool: Inputdevice, DeviceKey
Returns true if the given device key was released during the current update, false oherwise.
DeviceWasKeyDownbool: Inputdevice, DeviceKey
Returns true if the given device key was down during the last device update, false otherwise.
