1. Management functions
PictureClonePicture: Picture source
Clone a picture object.
PictureNewPicture:
Create a new picture object.
2. Drawing functions
PictureAllocbool: Picture, width, height
Allocate the picture buffer.
PictureApplyConvolutionvoid: Picture, int kernel_width, int kernel_height, array kernel_values, float filter_weight, int pass_count
Apply a convolution filter to the picture using a user specified kernel of values.
The weight and pass parameters will be 1 in most use case.
Kernel values are integer in the nominal range [0;255].
PictureFillvoid: Picture, Vector rgba
Fill the picture with a solid color defined from an RGBA vector.
PictureFillLockAlphavoid: Picture, Vector rgba
Fill the picture with a solid color defined from an RGBA vector, does not write to alpha.
PictureFillRectvoid: Picture, Vector rgba, Rect
Fill a rectangle inside the picture with a solid color defined from an RGBA vector.
PictureGetPixelVector: Picture, float x, float y
Return the picture pixel at coordinate {x, y} in picture space as a RGBA vector.
PictureGetRectRect: Picture
Return the picture rectangle in picture space.
PictureLinebool: Texture, float start_x, float start_y, float end_x, float end_y, Vector rgba
Draw a line, color is specified as an RGBA vector. Alpha is supported.
PictureLoadContentbool: Picture, string path
Load picture content from file.
PictureSaveTGAvoid: Picture, string path
Save picture content to Targa.
PictureSetPixelvoid: Picture, float x, float y, Vector rgba
Set picture pixel at coordinate {x, y} in picture space from an RGBA vector.
PictureWriteTextvoid: Picture, rect, string text, Font font, TextState
Render formatted text to a picture.
3. Manipulation functions
PictureFlipvoid: Picture, bool horizontal, bool vertical
Flip picture on one or both axis.
PictureResizevoid: Picture, float w, float h
Resize picture.
4. Blitting functions
PictureBlitvoid: Picture source, Picture destination, BlendMode mode
Blit a picture to another picture, the blend mode can be specified.
PictureBlitRectvoid: Picture source, Picture destination, Rect source, Rect destination, BlendMode mode
Blit a picture to another picture, both clipping zones and the blend mode can be specified.
PictureBlitRectMaskedvoid: Picture source, Picture destination, Picture mask, Rect source, Rect destination
Blit a picture to another picture using a third picture alpha channel as an opacity mask.
