Wii Binding Reference
1. Wii Home menu
WiiEnableHomeMenuvoid: bool
Enable or disables the home button.
WiiGetHomeMenuRunningbool: void
Returns the state of the Home menu.
WiiReturnToDataManagervoid: void
Launches the Home menu (only if it is enabled).
WiiStartHomeMenuvoid: void
Launches Home menu (only if it is enabled).
2. Wii strap screen functions
WiiDeInitStrapvoid: void
Uninitialize strap screen data.
WiiDrawStrapvoid: void
Draws strap screen data.
WiiInitStrapvoid: void
Initializes strap screen data.
3. Wii system functions
WiiEnableResetButtonvoid: bool
Enable or disables the reset/shutdown button.
WiiGetLastErrorStrIdstring: void
Returns ID of the last error, returns an empty string if none.
WiiGetLastErrorTextstring: void
Returns a localized string describing last error.
WiiGetOptionTextstring: string error_id
Returns a localized string describing a non blocking error option string ID.
WiiGetSquareRatioFor16_9float: void
Returns the ratio to use to scale 2D images in order to have square pixels on any TV screen.
WiiGetVersionstring: void
Returns "NOE" (Europe) or "NOA" (America).
WiiIs16_9bool: void
Returns true if the Wii system configuration is set to 16/9 mode.
WiiSetGameTitlevoid: string title
Sets the game title that may be displayed in Wii error screens
