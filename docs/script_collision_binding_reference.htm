1. Collision Shape Type functions
ShapeSetBoxbool: Colshape shape, Vector dimensions
Set the shape as a box collision shape.
ShapeSetCapsulebool: Colshape shape, float radius, float length
Set the shape as a Z-oriented capsule collision shape.
ShapeSetConvexbool: Colshape shape, string path
Set the shape as a convex collision.
ShapeSetCylinderbool: Colshape shape, float radius, float length
Set the shape as a Z-oriented cylinder collision shape.
ShapeSetMeshbool: Colshape shape, string path
Set the shape as a collision mesh.
ShapeSetSpherebool: Colshape shape, float radius
Set the shape as a sphere.
2. Collision Shape functions
ShapeGetItemItem: Colshape shape
Get the item this collision shape belongs to.
ShapeGetMassfloat: Colshape shape
Get the collision shape mass.
ShapeGetPositionVector: Colshape shape
Get the collision shape position in item space.
ShapeGetRotationVector: Colshape shape
Get the collision shape rotation as an Euler triplet in item space.
ShapeSetFrictionvoid: Colshape shape, float dynamic_friction, float static_friction
Set collision shape dynamic and static friction.
ShapeSetMassvoid: Colshape shape, float mass
Set the collision shape mass. Do not forget to update the item collision setup to update changes.
ShapeSetPositionvoid: Colshape shape, Vector position
Set the collision shape position in item space.
ShapeSetRestitutionvoid: Colshape shape, float restitution
Set collision shape restitution (1 for a perfect elastic collision).
ShapeSetRotationvoid: Colshape shape, Vector position
Set the collision shape position from a Euler triplet in item space.
3. Item functions
ItemAddCollisionShapeColshape: Item
Add a new collision shape to item.
ItemCollisionActivatevoid: Item, bool active
Activate or deactivate item collision.
ItemGetShapeFromIndexColshape: Item, int index
Get item collision shape from index.
ItemSetCollisionMaskvoid: Item, int collision_mask
Set item collision mask, this mask is a bit field.
ItemSetSelfMaskvoid: Item, int self_mask
Set item self mask, this mask is a bit field.
