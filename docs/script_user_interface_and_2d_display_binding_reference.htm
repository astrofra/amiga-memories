2. UI Font Settings
UIFontComputeRectrect: rect clip_rect, string text, Font font, table param
Compute the bounding rect of a formatted text string, does not perform any graphic output.
The following table keys are available:
  • 'color': Hexadecimal RGBA (eg. Red: xff0000ff)
  • 'align': TextAlign
  • 'format': TextFormat
  • 'tracking': Integer value specifying an extra space between glyphs.
  • 'heading': Integer value specifying an extra space between lines.
UIFontSetFallbackvoid: Font font, Font fallback
Set a font to query when a glyph is missing from this font.
UIFontSetParametersOffsetvoid: Font, float size_multiplier, float tracking_offset, float leading_offset
Set font-specific properties offset. This function is usually used to normalize font characteristics when porting an application to a different locale.
